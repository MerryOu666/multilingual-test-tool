# 多语种测试用例生成工具

按语种一键批量生成原生测试用例，支持按数量或按目标字数（100/500/1000/2000）两种模式生成，支持导出 Excel / CSV。

**在线使用：** https://multilingual-test-tool-9rsisz5uouyxnfs6m27jh6.streamlit.app

## 支持语种

中文、英语、泰语、越南语、韩语、法语、日语、西班牙语、阿拉伯语、俄语、葡萄牙语、德语、菲律宾语、马来语、印尼语（共15种）

## 用例类别

| 类别 | 说明 |
|------|------|
| 常规文案 | 日常通知、新闻、对话 |
| 长难句 | 复杂语法、多从句结构 |
| 短文本 | 按钮、标签等1-5字文案 |
| 数字日期 | 本地化数字、日期、货币格式 |
| 特殊字符 | 标点、emoji、公式混排 |
| 边界场景 | 超长文本、空白符、注入测试 |
| 专业术语 | 法律/医学/技术/金融领域 |
| 口语俚语 | 网络用语、非正式表达 |

## 使用方法

1. 打开上方链接
2. 选择语种 → 选择生成模式（按数量 / 按字数）→ 设置数量或目标字数 → 点击生成
3. 下载 Excel 或 CSV 文件

## 本地运行

```bash
pip install streamlit openpyxl
streamlit run app.py
```
